このサイトは、実際に使ったことのあるスクリプトを記録・公開するための個人サイトです。

見た目は Copper File をベースに、書類棚や保管庫のように落ち着いて積み上げていける雰囲気にしています。

## カテゴリ

- **PowerShell** — 日常作業や自動化で使ったスクリプト
- **いたずら用** — 軽い遊びやネタ用途のスクリプト
- 記事が増えた段階で、カテゴリを追加していく予定です

## デザイン方針

- 派手さよりも、保管しやすさと一覧性を優先
- 長期運用しても古びにくい、硬めの記録庫らしい見た目を維持

## このサイトで重視すること

- 後から見ても再利用しやすいこと
- スクリプトの目的と実行方法が明確であること
- 公開して問題のない内容だけを載せること

## 掲載しないもの

- API キーや認証情報を含むもの
- 業務固有の情報を含むもの
- 再現が難しすぎるもの
- 第三者に誤用されやすいもの
